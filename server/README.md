## Getting Started

1. Clone the repository:
    ```bash
    git clone ........
    ```
2. Navigate to the backend directory:
    ```bash
    cd Plasma_Full-Stack_App/server
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Start the server:
    ```bash
    nodemon server.js
    ```
